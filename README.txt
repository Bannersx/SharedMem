-----------------------------------------------------------------------Intrucciones para ejecutar el proyecto-----------------------------------------------------------------------

1) Abra una consola en la terminal y ubíquese en el directorio donde está el código del proyecto, en este directorio se verá un archivo llamado "Makefile"
2) En caso de ejecutarlo por primera vez, escriba en la terminal  "Make libraries" y presione enter
3) Para crear un inicializador, escriba "Make initializer" y presione enter
4) Para crear un productor, escriba "Make producer" y presione enter,este crea un productor con valores por defecto
5) Para crear un consumidor, escriba "Make consumer" y presione enter, este crea un consumidor con valores por defecto
6) Para crear un finalizador, escriba "Make finalizer" y presione enter, este terminará todos los procesos
